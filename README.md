# California House Price Prediction Web App

## Overview
End-to-end machine learning project to predict house prices using multiple features. 
Includes data preprocessing, model training, and cloud deployment using Flask.

## Tech Stack
- Python
- Pandas, NumPy, Scikit-learn
- Flask
- HTML/CSS
- Render (Cloud Deployment)

## Features
- Data cleaning and feature engineering
- Regression model training
- Web interface for user input
- Deployed ML model for real-time predictions

## How to Run
pip install -r requirements.txt  
python app.py
